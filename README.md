# Investment Simulator

A mobile-first investment learning simulator with a CNY virtual account, market-linked ETF price snapshots, simulated orders, slippage, fee details, pending orders, and portfolio tracking.

This is for education and simulation only. It does not place real trades and is not financial advice.

## GitHub Pages

The app is served from `docs/`.

Quote snapshots are stored in `docs/quotes.json`.

## Local Optional Server

`server.ps1` can serve the app locally and proxy live Yahoo chart data during local testing.
